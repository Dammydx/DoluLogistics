## Features

- Real-time parcel tracking and status updates
- Customer support messaging and request handling
- Admin dashboard to manage parcels, requests and messages
- Responsive, mobile-first UI
- Secure authentication and role-based access

## Tech Stack

- Frontend: React + Vite + TypeScript
- Styling: Tailwind CSS
- Backend & Database: Supabase
- Animations: Framer Motion
- Icons: Lucide React
- Notifications: React Toastify

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-org/dolu-logistics.git
   cd dolu-logistics
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Copy environment example and add Supabase credentials:
   ```bash
   cp .env.example .env
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
